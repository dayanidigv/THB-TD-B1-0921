Var, Let, Const - Simple Examples

What are they?

Three ways to make variables in JavaScript:

  • var - Old way (don't use)
  • let - Good for things that change
  • const - Good for things that don't change

Examples:

Look at the console to see the examples working!

Rule 1: const cannot change

const name = "John" - name will always be "John"

Rule 2: let can change

let age = 10 - age can become 11, 12, etc.

Rule 3: var is old and confusing

Don't use var. Use let and const instead.
