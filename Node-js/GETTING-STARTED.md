## Running Examples
```bash
# Basic examples
node 01-Introduction/02-hello-world.js
node 03-File-System/01-reading-files.js

# HTTP servers
node 04-HTTP-Server/01-basic-server.js  # Port 3000
node 04-HTTP-Server/04-express-basic.js # Port 3002

# MongoDB (start MongoDB first!)
mongod
node 06-APIs-and-Databases/03-mongodb-basics.js
```

## Installation
```bash
# For Express examples
cd 04-HTTP-Server
npm install express

# For MongoDB examples
cd 06-APIs-and-Databases
npm install mongodb mongoose

# For Todo API project
cd 07-Real-Projects/01-todo-api
npm install
npm start  # Port 3003
```
